Anthropic于2026年3月12日推出“Claude Certified Architect”认证，并开放了13个自学课程，全部免费、无订阅、无付费墙、无需注册，平台入口：（[https://anthropic.skilljar.com](https://t.co/NpV9hPAYXO)。考试本身99美元，但准备阶段零成本。
以下按12周路线图顺序，每个课程都附上官方直达链接（点击即可进入）：
Weeks 1-2：基础阶段（Foundations）
1. Claude 101（Anthropic Academy，免费） ![🔗](https://abs-0.twimg.com/emoji/v2/svg/1f517.svg "Link symbol") [https://anthropic.skilljar.com/claude-101](https://t.co/eRFkvKhHNk)
说明：涵盖Claude的使用模式、能力范围、Projects、Artifacts、Skills、Connectors以及实际工作流。即使你已经用Claude几个月，也建议从这里开始——考试全程都假设你掌握这些基础知识。
2. AI Fluency: Framework and Foundations（Anthropic Academy，免费，与大学教授联合开发） ![🔗](https://abs-0.twimg.com/emoji/v2/svg/1f517.svg "Link symbol") [https://anthropic.skilljar.com/ai-fluency-framework-foundations…](https://t.co/q1N5VdlbFJ)
说明：讲解有效AI协作的四大原则：delegation（委托）、description（描述）、discernment（辨别）、diligence（勤奋）。这是人类与AI合作的理论框架，帮助建立正确的心态。
Weeks 3-5：Claude API应用构建
3. Building Applications with the Claude API（Anthropic Academy，免费，8.1小时） ![🔗](https://abs-0.twimg.com/emoji/v2/svg/1f517.svg "Link symbol") [https://anthropic.skilljar.com/claude-with-the-anthropic-api…](https://t.co/Lcj79DGvqO)
说明：这是旗舰课程！从基础API调用到高级agentic架构、RAG管道全覆盖，包括系统提示、工具调用、流式传输、认证、结构化输出、上下文管理、生产模式等。直接对应考试Domain 1、4、5。 补充免费资源：Anthropic官方API文档（[https://docs.anthropic.com](https://t.co/XAUDrYXyHs)）
Weeks 6-7：Model Context Protocol（MCP工具协议）
4. Introduction to Model Context Protocol（Anthropic Academy，免费） ![🔗](https://abs-0.twimg.com/emoji/v2/svg/1f517.svg "Link symbol") [https://anthropic.skilljar.com/introduction-to-model-context-protocol…](https://t.co/ZVgwfRVzTg)
说明：MCP基础知识（工具、资源、提示），直击考试Domain 2（Tool Design & MCP Integration）。
5. Model Context Protocol Advanced（Anthropic Academy，免费） ![🔗](https://abs-0.twimg.com/emoji/v2/svg/1f517.svg "Link symbol") [https://anthropic.skilljar.com/model-context-protocol-advanced-topics…](https://t.co/PEIPTtIi4g)
说明：用Python深度构建MCP服务器和客户端，包含采样、通知、文件系统访问、生产传输机制等实战内容。 补充免费资源：MCP官方规范文档（[https://modelcontextprotocol.io](https://t.co/dJCxUNAlCe)）
Weeks 8-9：Claude Code开发工作流
6. Claude Code in Action（Anthropic Academy，免费） ![🔗](https://abs-0.twimg.com/emoji/v2/svg/1f517.svg "Link symbol") [https://anthropic.skilljar.com/claude-code-in-action…](https://t.co/FUkuY8IfAr)
说明：Claude Code完整配置，包括CLAUDE.md层级结构、路径特定规则、自定义slash commands、Skills、Hooks，以及CI/CD集成。
7. Claude Code on Coursera（Vanderbilt University，免费审计） ![🔗](https://abs-0.twimg.com/emoji/v2/svg/1f517.svg "Link symbol") [https://coursera.org/specializations/generative-ai-software-engineering…](https://t.co/HcOD64N8gv) （具体模块：Claude Code: Software Engineering with Generative AI Agents，由Dr. Jules White教授主讲）
说明：属于“Generative AI Software Engineering Specialization”课程。聚焦AI-first开发和“Best of N”模式，提供与Anthropic官方课程不同的视角。（免费审计，无需付费即可学习）
8. Building Skills for Claude Code（Anthropic Academy，免费） ![🔗](https://abs-0.twimg.com/emoji/v2/svg/1f517.svg "Link symbol") [https://anthropic.skilljar.com/introduction-to-agent-skills…](https://t.co/Gwm36wLwG5)
说明：专门教如何创建、配置、分发可复用的Skills，直接对应考试中“生产级工作流”要求。
